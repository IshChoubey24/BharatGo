# BharatGo - E-Commerce Platform

# Installation

1. Clone the repository:
```bash
git clone  git@github.com:IshChoubey24/BharatGo.git
cd bharatgo
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open in your browser.
